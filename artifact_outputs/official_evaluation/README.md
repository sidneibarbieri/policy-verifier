# Official Aggregate Evaluation Outputs

This directory preserves the aggregate outputs from the canonical paid full-corpus run used by the paper.

- `summary.json`: top-level aggregate metrics for the official paid run.
- `analysis_bundle.json`: typed aggregate bundle with by-model, by-rule-treatment, and pairwise results.
- `official_checklist.json`: readiness checklist emitted for the official paid run.

These files are included so reviewers can compare the reported paper results against a stable public aggregate snapshot.

The raw paid execution lineage, prompt payloads, and per-run snapshots are intentionally excluded from the public artifact.
